MotoKey Hardware Architecture

MVP design with real components. Goal: build cheap, sell profitably at $89 / $149.

1 System Overview

The whole system is three things: a lock box on the bike, a key fob in your pocket, and a security protocol binding them together. That's it.

Key Design Decision:

One-way RF (fob transmits, box receives). This is half the cost of two-way communication and the fob battery lasts 3x longer. The tradeoff: no confirmation beep from the bike. The LED on the lock box blinks instead. For $89, this is the right call.

4 Security Protocol

This is what prevents someone from cloning the fob or replaying a captured signal.

Pairing (one-time setup)

  FOB                                    LOCK BOX
   │                                        │
   │  1. User presses PAIR button on box    │
   │         LED blinks fast (green)        │
   │                                        │
   │  2. User presses FOB button            │
   │  ──── plaintext: "PAIR_REQUEST" ─────► │
   │                                        │
   │  3. Box generates random 128-bit key   │
   │     Box stores: KEY + FOB_ID           │
   │     Box shows LED pattern:             │
   │     (4 blinks = key byte checksum)     │
   │                                        │
   │  4. Box transmits key via IR/NFC       │
   │  ◄──── one-time key transfer ────────  │  ← requires physical contact
   │                                        │
   │  5. Fob stores KEY in EEPROM           │
   │     Both set counter = 0               │
   │                                        │
   │  DONE. Shared secret established.      │
   └────────────────────────────────────────┘

   NOTE: Key transfer via NFC/physical contact means an attacker
   cannot intercept the pairing from a distance. You must be
   holding the fob against the box.

Unlock (every use)

  FOB                                    LOCK BOX
   │                                        │
   │  1. User presses button                │
   │     Fob increments counter             │
   │     Fob computes:                      │
   │       payload = counter | FOB_ID       │
   │       encrypted = AES-128(KEY, payload)│
   │                                        │
   │  ──── 433MHz: encrypted packet ──────► │
   │       (16 bytes, ~3ms air time)        │
   │                                        │
   │  2. Box decrypts with stored KEY       │
   │     Checks FOB_ID matches              │
   │     Checks counter > last_counter      │
   │     If valid:                          │
   │       - Update last_counter            │
   │       - Close relay (power ON)         │
   │       - Start 30-second timeout        │
   │       - LED solid green                │
   │                                        │
   │  3. After 30 sec with no new signal:   │
   │     - Open relay (power OFF)           │
   │     - LED blink red                    │
   │                                        │
   └────────────────────────────────────────┘

Why this is hard to hack

Replay Attack

Attacker records a valid transmission and replays it. Fails because the counter has already advanced. The lock box rejects any counter ≤ last seen.

Brute Force

AES-128 has 2128 possible keys = 3.4 × 1038. At 1 billion attempts/second, it would take 1022 years. The universe is 1.4 × 1010 years old.

Signal Analysis

Every transmission looks like random noise. No pattern emerges even with millions of captured packets. AES is the same encryption used by the US government for classified data.

Physical Bypass

Relay is normally-open. Cutting wires = no power. Cracking the potted enclosure destroys the PCB. The only bypass is splicing around the entire box, which requires stripping the bike apart.

9 Scaling Considerations

10 units (garage batch)

Hand-solder perfboard. Buy parts from Amazon. COGS ~$30/unit. Margin ~66% at $89.

100 units (first real run)

Custom PCB from JLCPCB ($0.80/ea). Parts from LCSC. COGS ~$23/unit. Margin ~74% at $89.

1000+ units

PCBA assembly (JLCPCB SMT). Custom injection mold enclosure. COGS drops to ~$15/unit. Margin ~83%.

Bottom line:

The Basic kit costs $23 to build at 100 qty and sells for $89. That's a 73% gross margin. Even selling 10 hand-built units at $30 COGS, you clear $59 per unit. The math works at every scale.
